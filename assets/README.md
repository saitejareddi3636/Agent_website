# Assets Directory

This directory contains all the assets for the Agent Website.

## Directory Structure

```
assets/
├── images/          # Image files (logos, illustrations, photos)
├── icons/           # Icon files (favicon, social media icons)
├── fonts/           # Custom fonts (if any)
└── documents/       # Documents like brochures, PDFs
```

## Usage

- **Images**: Add product images, hero illustrations, team photos, etc.
- **Icons**: Include favicon.ico, apple-touch-icon.png, and other icon files
- **Fonts**: Place custom font files here if not using web fonts
- **Documents**: Store downloadable resources like whitepapers, brochures

## Image Guidelines

- Use WebP format for modern browsers with fallbacks
- Optimize images for web (compress file sizes)
- Use descriptive filenames
- Include alt text in HTML for accessibility

## Recommended Image Sizes

- Hero image: 1200x600px
- Service icons: 64x64px or SVG
- Team photos: 300x300px
- Logo: SVG format preferred
