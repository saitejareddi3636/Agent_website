## Getting Started

### Prerequisites
- A modern web browser
- A local web server (optional, for development)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/saitejareddi3636/Agent_website.git
   cd Agent_website
   ```

2. **Open the website**
   - For development: Open `index.html` directly in your browser
   - For local server: Use any local server like Live Server, Python's HTTP server, or Node.js serve

3. **Using Python HTTP Server**
   ```bash
   python -m http.server 8000
   # Then open http://localhost:8000 in your browser
   ```

4. **Using Node.js serve**
   ```bash
   npx serve .
   # Follow the URL provided in the terminal
   ```

## Customization

### Colors
The website uses a blue color scheme. To change the primary color, update the CSS variables in `css/style.css`:

```css
/* Change #3b82f6 to your preferred color throughout the file */
.btn-primary {
    background-color: #your-color;
}
```

### Content
Update the content in `index.html`:
- Company name and branding
- Service descriptions
- Contact information
- About section content

### Images
Add your images to the `assets/images/` directory and update the references in the HTML.
